# CampusTourGuide

一个基于 **C/C++** 和 **Visual Studio** 的校园导游系统，使用控制台界面展示校园景点信息，并支持查询景点详情、查看校园地图和计算景点间最短路径。

## 功能介绍

- 查看校园地图
- 显示所有景点列表
- 查询景点详细信息
- 查询景点间最短路径
- 查看邻接矩阵形式的路径数据
- 支持中文输出和图标展示
- 使用 **Dijkstra** 算法计算最短路径

## 项目特点

- 采用邻接矩阵存储校园景点图结构
- 内置 10 个校园景点数据
- 控制台交互式菜单，操作简单
- 支持打开校园地图图片
- 适合《数据结构与算法》课程设计参考

## 运行环境

- Windows
- Visual Studio 2022 或更高版本
- C/C++ 开发工具集
- 支持 UTF-8 的控制台环境

## 项目结构

- `CampusTourGuide.sln`：解决方案文件
- `CampusTourGuide.vcxproj`：项目文件
- `main.c`：程序入口与菜单交互
- `graph.c`：图结构、景点数据与路径算法实现
- `graph.h`：图结构与函数声明
- `icons.h`：控制台图标定义
- `Campus_map.jpg`：校园地图图片

## 构建方法

1. 使用 Visual Studio 打开 `CampusTourGuide.sln`
2. 选择需要的配置，如 `Debug | x64`
3. 点击“生成解决方案”
4. 运行程序

## 使用说明

启动后，程序会显示主菜单：

1. 查看校园地图  
2. 显示所有景点列表  
3. 查询景点详细信息  
4. 查询景点间最短路径  
5. 查看详细路径数据（邻接矩阵）  
0. 退出系统  

## 地图图片说明

程序会尝试查找并打开以下图片文件之一：

- `campus_map.jpg`
- `campus_map.png`
- `map.jpg`
- `map.png`
- `校园地图.jpg`
- `校园地图.png`

如果未自动打开，请手动在项目目录中查看图片文件。

## 算法说明

最短路径查询功能使用 **Dijkstra 算法**，适用于带权无向图场景。
